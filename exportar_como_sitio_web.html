Exportar como Sitio web

Esta opción permite exportar el proyecto como un sitio web listo para ser alojado en un servidor de Internet. Además de las páginas desarrolladas, se creará un menú de navegación lateral que reproduce el árbol de contenidos del objeto.

HTML es el lenguaje con el que se escriben las páginas web y es el formato de exportación más sencillo que se puede realizar con eXe. Las páginas HTML generadas se pueden subir a sitios web (a un servidor), grabarlas en una memoria USB, enviarlas por correo... Todo con la tranquilidad de que nuestros alumnos podrán ver el recurso, pues todos los ordenadores personales disponen de navegador web para abrir las páginas generadas.

Al exportar como Sitio Web, eXeLearning crea una carpeta donde le indiquemos, y dentro de esa carpeta, coloca todos los archivos (imágenes, textos, etc.) que hayamos utilizado en nuestro proyecto.

PASOS

Los pasos a seguir para exportar como sitio web serán:

  1. Abrimos el archivo .elp que queremos exportar
  2. Hacemos clic sobre el botón Archivo/Exportar/Sitio web
  3. Nos situamos en la ruta donde queremos guardar la carpeta que se generará y pulsamos sobre el botón Guardar

En la ruta indicada se generará una carpeta que contendrá todos los archivos necesarios. Para visualizar el recurso, basta con entrar en la carpeta y abrir el archivo index.html que eXeLearning habrá creado en su interior. Al hacer doble clic en este archivo el recurso se abrirá en el navegador que utilice el equipo por defecto. Recordamos en todo caso que se trata de una carpeta localizada en el equipo, por lo que, aunque se despliegue en el navegador, no está utilizando ningún tipo de conexión a internet.

Archivo index.html dentro de carpeta creada

Si lo que queremos es subir este contenido a internet, debemos publicarlo, paso que se explica en el apartado siguiente. 

Si activamos el Modo Avanzado, comprobamos que se ofrecen 2 opciones más de exportación como sitio web:

  • Carpeta auto-contenida: Es la opción "Sitio web" ya comentada que aparece por defecto en el Usuario básico. Crea una carpeta donde le indiquemos, y dentro de esa carpeta, coloca todos los archivos (imágenes, animaciones, etc.) que hayamos utilizado en nuestro proyecto. Para reproducirlo, basta con abrir el archivo index.html que eXeLearning habrá creado en su interior. Se mostrará una página web con una tabla de contenidos navegable a la izquierda, y si pulsamos sobre un enlace de esa tabla se mostrará en pantalla dicha sección. Podemos grabar el contenido de la carpeta donde deseemos (memoria USB), subirlo a una página web, etc. Y así poder entregar los apuntes a los alumnos.
  • Archivo comprimido ZIP: Genera una estructura igual a la del punto anterior, pero comprimiendo en un único archivo ZIP todos los ficheros que estarían en una carpeta auto-contenida como la anterior. Tan solo debemos indicar el nombre del fichero y automáticamente le asignará la extensión .zip. Es ideal para enviar por correo electrónico o enviar por Internet.
  • Página HTML única: Crea también una carpeta con todos los recursos necesarios, pero en lugar de generar varias páginas web y una tabla o índice de contenidos, crea una única página web, muy larga, donde todos los contenidos se van mostrando unos detrás de otros. Es útil para imprimir el contenido.
